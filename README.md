# Heart QR Code Generator

A simple web app to generate heart-shaped QR codes.

## Development

### Prerequisites

- Node.js >= 23
- pnpm

### Installation

```sh
pnpm install
```

### Dev Server

```sh
pnpm dev
```

### Build

```sh
pnpm build
```

### OG Image

Regenerate the Open Graph image (`public/og.png`):

```sh
node scripts/generate-og.mjs
```

The Pacifico font is downloaded automatically on first run.
